# Weather App

A modern weather application built using React and Material UI.  
This project fetches real-time weather data using the OpenWeather API and displays weather information with a clean UI, smooth animations, and responsive component rendering.

## Features

- Real-time weather data fetching
- API handling using OpenWeather API
- Dynamic component rendering
- Search weather by city


## Tech Stack

- React.js
- Material UI
- CSS3
- OpenWeather API
